# Lectures conseillees

- Alan Kay, textes et conferences sur Smalltalk : pour comprendre l'idee d'objets qui communiquent par messages.
- Grady Booch, *Object-Oriented Analysis and Design with Applications* : référence théorique sur objets, classes, responsabilités et collaborations.
- Documentation TypeScript sur les classes : utile pour comparer le modèle objet du langage utilisé dans l'app.
