# Construire l'ouverture d'un cours

## Objectif

Rédiger le premier chapitre d'un autre corpus en respectant la même logique :

- histoire courte ;
- auteurs clés ;
- bibliographie de départ ;
- lien explicite avec les notions à venir.

## Travail attendu

Écrire une ouverture de 400 à 700 mots pour un autre sujet pédagogique. Le texte doit présenter :

1. l'origine du sujet ;
2. quatre à six acteurs clés ;
3. les concepts qu'ils ont rendus plus clairs ;
4. la manière dont le reste du cours va s'en servir.

## Critères

- le texte reste lisible ;
- les auteurs sont reliés à des idées concrètes ;
- la bibliographie n'est pas décorative ;
- l'ouverture ne remplace pas le cours, elle le prépare.
