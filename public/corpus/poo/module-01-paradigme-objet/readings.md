# Lectures conseillees

- Alan Kay, textes et conferences sur Smalltalk : pour comprendre l'idee d'objets qui communiquent par messages.
- Grady Booch, *Object-Oriented Analysis and Design with Applications* : reference theorique sur objets, classes, responsabilites et collaborations.
- Documentation TypeScript sur les classes : utile pour comparer le modele objet du langage utilise dans l'app.
