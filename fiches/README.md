# Fiches personnelles

Ajoute une fiche en créant un fichier Markdown dans ce dossier, puis référence-la dans `index.json`.

Format minimal :

```md
---
id: fiche.mon-sujet
title: Mon sujet
tags:
  - def
  - important
---

# Mon sujet

Ma synthèse personnelle.
```

Les seuls tags visibles dans l'application sont `def` et `important`.
